Práctica: Listas en HTML

En esta actividad, practicarás la creación de listas ordenadas, no ordenadas y anidadas utilizando HTML. Completa las tareas indicadas en los comentarios del código y utiliza las etiquetas <ol>, <ul>, y <li> para resolverlas.

Instrucciones

  1. Crea una lista ordenada con tus tres películas favoritas.
  2. Crea una lista no ordenada con tres hobbies que disfrutes.
  3. Crea una lista anidada dentro de una lista ordenada:
    • El primer nivel debe enumerar dos tipos de comida (ejemplo: italiana, japonesa).
    • Cada tipo de comida debe incluir al menos dos platos típicos como subelementos.
  4. Utiliza atributos como type y start en al menos una de las listas ordenadas.
  5. Utiliza diferentes estilos de viñeta en las listas no ordenadas aplicando CSS (list-style-type).
